# Molvi — фирменная палитра и лендинг

Дата: 2026-07-09

## Цель

1. Зафиксировать фирменную палитру Molvi (выведена из логотипа).
2. Привести цвета в коде приложения к этой палитре.
3. Сделать современный тёмный лендинг с кнопкой «Скачать».

## Палитра (источник — логотип: charcoal-фон, кремовый микрофон, коралловые волны)

| Токен | Hex | Назначение |
|---|---|---|
| `coral` | `#F04A3F` | акцент, CTA, запись, ссылки, фокус |
| `coral-hover` | `#FF5C50` | ховер акцента |
| `coral-deep` | `#C63A30` | нажатие/тени |
| `ink-900` | `#141316` | самый тёмный фон страницы |
| `ink-800` | `#1C1A1F` | фон окна/оверлея (matte) |

| `recording` | `#F04A3F` | оверлей: запись (= coral) |
| `transcribing` | `#4A5568` | оверлей: распознавание (сланец) |
| `warn` | `#E0A83E` | пауза (янтарь) |
| `success` | `#3FB27F` | «готов»/успех |

## Часть 1 — приложение

- Новый модуль `molvi/theme.py` — единый источник палитры (Python-константы).
- `tray.py`: иконка «готов» → `coral`, «пауза» → `warn` (вместо зелёного/оранжевого).
- `overlay.py`: fallback-текст и matte → токены (`recording`, `transcribing`, `ink-800`).
- `gen_assets.py`: цвета плейсхолдеров → брендовые (чтобы повторный запуск не затирал
  Gemini-ассеты старыми красным/синим).
- Overlay-PNG (`recording.png`, `transcribing.png`) уже на бренде — не трогаем.
- Нативные ttk-диалоги (`settings.py`, `wizard.py`) оставляем нативными: пиксельная
  перекраска native-виджетов на Windows рискует выглядеть хуже. Осознанное решение.

## Часть 2 — лендинг

- Стиль: тёмный, как приложение; референс — wisprflow.ai; современный.
- Позиционирование (отличие от Wispr Flow): **100% локально, приватно, бесплатно,
  без облака и подписки**. Никаких заявлений о фичах, которых нет (AI-правка, облако,
  мобильные, синхронизация) — Molvi это локальная сырая транскрипция Whisper.
- Секции: hero (тэглайн + CTA + «для Windows 10/11») → дифференциаторы (локально/
  бесплатно/open-source) → как работает (3 шага) → сетка фич → требования → футер
  (GitHub, MIT).
- Технически: статичный HTML/CSS, самодостаточный, папка `docs/site/`. Деплой — на свой
  сервер (без привязки к GitHub Pages). Кнопка «Скачать» → GitHub releases/latest,
  адрес — одной правимой константой.
- Дизайн-скилы: `ui-ux-pro-max`, `frontend-design`.
- Язык: русский.

## Вне scope

- Перерисовка логотипа/иконки (уже финальные).
- Полная тема нативных ttk-диалогов.
- CI/деплой сайта (пользователь задеплоит сам).

NAME
molvi — говоришь, а не печатаешь
SYNOPSIS
удерживай Ctrl (настраивается) → говори → отпусти
ENGINE
faster-whisper / large-v3 на CUDA; CPU-fallback на base
NETWORK
ноль байт наружу. модель качается один раз, дальше офлайн

LICENSE

cat features.txt

  • печатает в терминал — распознаёт консоль и шлёт посимвольно, где Ctrl+V мёртв
  • идеален для Claude Code / Codex: длинный промпт за секунды, голосом
  • живой эквалайзер: волна ходит от твоего голоса — видно, что мик слышит
  • любой хоткей, любой микрофон, ru/en/auto
  • оверлей на мониторе с активным окном (multi-monitor — так, как надо)
  • автозапуск в трее, ~0% CPU пока молчишь

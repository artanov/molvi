cat features.txt

  • не крадёт фокус: пилюля появляется поверх, а курсор остаётся там, куда печатаем
  • идеален для Claude Code / Codex: длинный промпт за секунды, голосом
  • нативный Apple Silicon: движок MLX, unified memory, ноль ватт впустую
  • живой эквалайзер: волна ходит от твоего голоса — видно, что мик слышит
  • правый ⌘ по умолчанию; любой хоткей, любой микрофон, ru/en/auto
  • живёт в строке меню, ~0% CPU пока молчишь

cat INSTALL.md

  1. качаешь Molvi.dmg (~150 МБ) — кнопка ниже, перетаскиваешь в Applications
  2. первый запуск: правый клик → Открыть # нотаризации нет — есть исходники
       альтернатива: xattr -cr /Applications/Molvi.app
  3. мастер скачает модель (large-v3-turbo ≈ 1.6 ГБ) и попросит два разрешения: Мониторинг ввода + Универсальный доступ
  4. зажал ⌘ → сказал → готово

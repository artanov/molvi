# Исходники ассетов

Сгенерированные картинки (Gemini), из которых получены runtime-ассеты
`molvi/assets/recording.png` и `molvi/assets/transcribing.png` (400×128, RGBA).

Этот каталог НЕ пакуется в дистрибутив — в `molvi/assets/` лежит только то,
что нужно приложению в рантайме (плюс `icon.png` для сборки иконки exe).
